üîß Gmail Setup (Recommended)

Step 1: Enable 2-Step Verification
  • Go to Google Account Settings
  • Click on "Security" in the left sidebar
  • Under "Signing in to Google", enable "2-Step Verification" if not already enabled
Step 2: Generate App Password
  • In the same Security section, click on "App passwords"
  • You may need to sign in again
  • Select "Mail" as the app
  • Select "Other (Custom name)" as the device
  • Enter "CRM System" as the device name
  • Click "Generate"
Step 3: Copy the App Password
  • Google will show you a 16-character password (like: abcd efgh ijkl mnop)
  • Copy this password (remove spaces if any)
  • Important: This is NOT your regular Gmail password!
Step 4: Update .env File
  • Open the backend/.env file
  • Update the following lines:
EMAIL_HOST=smtp.gmail.com EMAIL_PORT=587 EMAIL_USER=your-gmail-address@gmail.com EMAIL_PASS=your-16-character-app-password
Step 5: Restart the Backend Server
  • Stop the backend server (Ctrl+C)
  • Start it again: python -m app.main
‚úÖ Test the Configuration:
  • Create a new user in the CRM system
  • Check the popup for email status
  • If successful, the user will receive their credentials via email

üîç Troubleshooting

Common Issues:
  • Authentication failed: Make sure you're using the App Password, not your regular password
  • 2-Step Verification required: Enable 2-Step Verification first
  • App Password not available: Make sure 2-Step Verification is fully set up
  • Connection timeout: Check your internet connection and firewall settings
  • Email not received: Check spam folder, verify email address is correct

üìã Current Configuration Status

http://localhost:8000/api/email/config/status

(Note: You need to be logged in as Super Admin to access this endpoint)

üß™ Test Email Sending

After configuration, you can test email sending by:
  • Creating a new user in the CRM system
  • The popup will show the email status
  • Green status = Email sent successfully
  • Red status = Email failed (check error message)
  • Blue status = Email queued for sending
  • Gray status = Email not configured
üîí Security Note: Never share your App Password or commit it to version control. The .env file should be kept private and not shared publicly.
